                                            CATALOGO DIGITAL

    Sistema de catálogo digital para pequenos comércios, permitindo o gerenciamento completo de produtos (CRUD).

Este projeto foi desenvolvido com foco em tecnicas de backend utilizando Node.js, Express e MongoDB, além de servir como base para uma futura aplicação comercial.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Tecnologias

* Node.js
* Express
* MongoDB
* Mongoose
* REST API
* JSON
* React-toastify

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Funcionalidades

* Criar produtos
* Listar produtos
* Atualizar produtos
* Deletar produtos

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Rotas da API

###  Criar produto

POST /produtos

###  Listar produtos

GET /produtos

###  Atualizar produto

PUT /produtos/:id

###  Deletar produto

DELETE /produtos/:id
